# maati-studio

Static multi-page marketing website for **Maati** — a full-service marketing and creative agency based in Bangalore.

**Stack:** [Astro](https://astro.build) (static output) + [Tailwind CSS v4](https://tailwindcss.com)
**Deploy target:** Cloudflare Pages
**Design language:** Apple.com-inspired — full-width sections, sharp corners, Playfair Display + Inter, earth-toned palette.

## Local development

```bash
npm install
npm run dev        # http://localhost:4321
npm run build      # static output → ./dist
npm run preview    # preview the production build
```

## Project structure

```
src/
├── layouts/BaseLayout.astro   Nav + Footer + scroll-reveal observer + fonts/SEO
├── pages/                     index, services, about, contact
├── components/                Nav, Footer, Hero, ServiceCard, StatBar, BriefForm, CtaBanner
├── data/services.ts           Single source of truth for the 9 service verticals
└── styles/global.css          Design tokens + base reset + typography system
public/images/                 Brand logos (Black / Bone / Earth)
scripts/make-logos.mjs         Regenerates placeholder wordmark PNGs
```

## Logos

`public/images/` currently holds **placeholder** wordmark PNGs generated by
`node scripts/make-logos.mjs`. Replace these with the real brand assets, keeping
the same filenames:

- `Maati_Logo_Maati_Black.png` — light backgrounds (nav)
- `Maati_Logo_Bone.png` — dark backgrounds (footer)
- `Maati_Logo_Earth.png` — accent / favicon

## Deploy — Cloudflare Pages

This is a fully static site, so connect the GitHub repo and use:

| Setting                | Value           |
| ---------------------- | --------------- |
| Framework preset       | Astro           |
| Build command          | `npm run build` |
| Build output directory | `dist`          |

The production site URL is set to `https://maati.studio` in `astro.config.mjs`
(drives the sitemap and canonical/OG tags) — update it there if the domain changes.
